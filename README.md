dotfiles
========

This is a very simple .dotfiles repo from a OS X beginner

## Installation

```bash
git clone https://github.com/kevinpapst/dotfiles.git && cd dotfiles && source bootstrap.sh
```

To update, change into your local `dotfiles` repository and then:

```bash
source bootstrap.sh
```
